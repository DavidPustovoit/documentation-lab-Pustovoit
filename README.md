# QuickShare - Сервіс обміну файлами

## Про проєкт
QuickShare — це веб-сервіс, який дозволяє користувачам швидко та безпечно обмінюватися файлами будь-якого формату. Головна особливість платформи — простота використання та автоматичне видалення файлів через заданий проміжок часу, що гарантує конфіденційність даних.

## Основні функції
* Завантаження файлів до 2 ГБ
* Генерація унікальних посилань для доступу
* Встановлення таймера життя файлу (від 1 години до 7 днів)
* Захист посилання паролем
* Перегляд статистики завантажень

## Для кого цей продукт
Цей сервіс створений для фрілансерів, студентів та офісних працівників, яким потрібно швидко передати великі файли без реєстрації та використання складних хмарних сховищ.
