# Інструкція з налаштування та запуску

Цей документ описує, як підготувати робоче середовище для використання локальної версії QuickShare (для розробників або тестувальників).

## Системні вимоги
* Сучасний веб-браузер (Chrome, Firefox, Safari)
* Доступ до мережі Інтернет
* Встановлений Node.js (версія 14 або вище) — для запуску сервера

## Процес встановлення

### 1. Клонування репозиторію
Відкрийте термінал та виконайте команду:
`git clone https://github.com/example/quickshare.git`

### 2. Встановлення залежностей
Перейдіть у папку проєкту та встановіть необхідні бібліотеки:
```
cd quickshare
npm install
```

### 3. Запуск локального сервера
Для старту програми введіть:
`npm start`

## Перевірка роботи
1. Відкрийте браузер.
2. Перейдіть за адресою `http://localhost:3000`.
3. Ви маєте побачити головну сторінку з логотипом QuickShare та кнопкою завантаження.
4. Спробуйте завантажити тестовий текстовий файл (.txt). Якщо з'явилося посилання — система працює коректно.
